# Sudoku Arena

Sudoku Arena - современная веб-платформа для Sudoku на Next.js и TypeScript. Мы сделали основу игрового продукта: ежедневные и свободные партии, интерактивную доску, заметки, откат хода, AI Coach, профиль игрока, Diamond-статус, алмазную экономику, серии дней, косметические темы, режим казахских орнаментов и глобальный рейтинг.

Продукт создан для игроков, которым нужен не просто старый сайт с головоломкой, а более живой игровой опыт: прогресс, соревнование, персонализация и понятные подсказки. Он также подходит как фундамент для будущего PvP-режима Sudoku Arena с матчами, сезонами, косметикой и соревновательной механикой.

Ценность проекта в том, что Sudoku превращается в современную игровую платформу: игроку интереснее возвращаться каждый день, развивать профиль, сравнивать результат с другими и учиться решать лучше через объясняющие подсказки, а не через простую выдачу ответа.

## Возможности

- Sudoku-доска с проверкой, заметками, откатом хода, клавиатурой и сохранением ежедневной партии.
- AI Coach с объяснениями для выбранной клетки и лимитом бесплатных подсказок.
- Diamond-режим с безлимитными подсказками, темами доски и режимом орнаментов.
- Алмазы, ежедневная серия, награды за решение и экран кошелька.
- Профиль игрока с аватаром, страной, городом, статистикой и рейтингом.
- Рейтинг Казахстана и глобальный рейтинг.
- Набор тестов для подписки, онбординга, экономики, рейтинга, орнаментов, серии дней и подсказок.

## Локальный запуск

```bash
npm install
npm run dev
```

Открой `http://localhost:3000`.

## Проверки


## Переменные окружения

Реальные API-ключи и токены не хранятся в Git. Даже для приватного репозитория безопаснее держать секреты локально или в GitHub Secrets.

После скачивания проекта скопируй шаблон:

```bash
cp .env.example .env.local
```

Заполни `.env.local` своими значениями:

```bash
NEXT_PUBLIC_SITE_URL=http://localhost:3000

# Опционально для постоянного хранения рейтинга
SUPABASE_URL=...
SUPABASE_SERVICE_ROLE_KEY=...
```

`.env.local` уже добавлен в `.gitignore`, поэтому секреты не попадут в коммит и не уйдут во фронтенд.
