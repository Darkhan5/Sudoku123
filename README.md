# Судоку

Игровая веб-платформа для судоку: ежедневная задача, свободная игра, ИИ-подсказки, подробный разбор партии, рейтинг Казахстана, профиль игрока, Судоку Пасс, алмазы, косметика и PvP-арена с другом по ссылке.

Production: https://sudoku123-theta.vercel.app

## Что умеет продукт

- Ежедневный судоку с сохранением прогресса, таймером, ошибками, подсказками и рейтингом.
- Свободная игра по сложности: лёгкая, средняя, сложная, эксперт.
- Удобная доска: заметки, откат хода, проверка поля, клавиатура, подсветка ошибок и доступные настройки контраста.
- ИИ-подсказки для выбранной клетки. У бесплатного игрока есть лимит, Судоку Пасс открывает безлимит.
- Game Review после завершения партии: оценка решения, слабые места, ключевые ходы, таймлайн и подробный анализ.
- Профиль игрока: имя, город Казахстана, аватар, статистика, серия дней, XP, уровни, ранги, титулы и достижения.
- Алмазы: награды за игру, серии, чистое решение и достижения; магазин наборов через Stripe Checkout.
- Судоку Пасс: сезон на 30 дней, бесплатный и премиум-трек, уровни, задания, темы, стили цифр, титулы и алмазы.
- Косметика: темы интерфейса, паки цифр, оформление профиля и режим казахских орнаментов `Өрнек`.
- Рейтинг: вкладки `Город` и `Казахстан`, один лучший результат игрока за день.
- PvP-арена: приватная комната по ссылке, одинаковая доска для двух игроков, прогресс друга и визуальные саботажи.

## Рейтинг

Места идут по очкам: чем больше очков, тем выше игрок. Формула:

```text
очки = 10 000 - время в секундах x 4 - ошибки x 360 - подсказки x 180
```

Минимум за решённую задачу — 1 000 очков. При равенстве выше игрок с меньшим временем, затем с меньшим числом ошибок и подсказок.

## Технологии

- Next.js 14
- React 18
- TypeScript

## Переменные окружения

Секреты не хранятся в Git. Для локальной разработки создай `.env.local` на основе `.env.example`.

Основные переменные:

```bash
NEXT_PUBLIC_SITE_URL=http://localhost:3000

STRIPE_SECRET_KEY=...
STRIPE_WEBHOOK_SECRET=...
STRIPE_SUDOKU_PASS_PRICE_ID=...
STRIPE_DIAMOND_STARTER_PRICE_ID=...
STRIPE_DIAMOND_SMALL_PRICE_ID=...
STRIPE_DIAMOND_MEDIUM_PRICE_ID=...
STRIPE_DIAMOND_LARGE_PRICE_ID=...

SUPABASE_URL=...
SUPABASE_SERVICE_ROLE_KEY=...
```

На Vercel эти значения нужно добавить в `Project Settings -> Environment Variables`. `SUPABASE_SERVICE_ROLE_KEY` должен использоваться только на сервере и не должен иметь префикс `NEXT_PUBLIC_`.

## Production-заметки

- Рейтинг и PvP локально могут работать через `.data`, но на Vercel нужно подключить Supabase.
- Stripe webhook принимает событие `checkout.session.completed` на `/api/stripe/webhook`.
- После покупки Судоку Пасса игрок возвращается к завершённой партии, и подробный Game Review не теряется.
